# Buchplan: Moderne Web-Entwicklung mit React, Vite & Node.js

## Für wen ist dieses Buch?
Senior-Entwickler mit .NET/MVC/Razor-Hintergrund, die moderne JavaScript/TypeScript-basierte Web-Entwicklung lernen möchten.

## Buchstruktur

### Teil I: Grundlagen & Paradigmenwechsel
1. **Kapitel 1: Von .NET MVC zu Modern JavaScript** - Der große Paradigmenwechsel
2. **Kapitel 2: Node.js & npm** - Die JavaScript-Runtime verstehen
3. **Kapitel 3: TypeScript** - Typsicherheit für JavaScript

### Teil II: Frontend-Stack
4. **Kapitel 4: React Fundamentals** - Komponenten, JSX, Props
5. **Kapitel 5: React Hooks & State** - useState, useEffect, Custom Hooks
6. **Kapitel 6: Vite** - Der moderne Build-Tool-Champion
7. **Kapitel 7: Tailwind CSS** - Utility-First CSS
8. **Kapitel 8: Zustand** - Einfaches State Management
9. **Kapitel 9: React Router** - Client-Side Routing

### Teil III: Backend-Stack
10. **Kapitel 10: Express.js** - Minimalistisches Backend-Framework
11. **Kapitel 11: SQLite & better-sqlite3** - Embedded Database
12. **Kapitel 12: REST API Design** - Moderne API-Patterns

### Teil IV: Das Gesamtbild
13. **Kapitel 13: Full-Stack Architektur** - Wie alles zusammenspielt
14. **Kapitel 14: Development Workflow** - Hot Reload, DevTools, Debugging
15. **Kapitel 15: Deployment & Production** - Build, Deploy, Hosting

### Teil V: Meta & Zukunft
16. **Kapitel 16: AI & Code-Generierung** - Warum Claude diese Technologien beherrscht
17. **Kapitel 17: Ökosystem & Alternativen** - Next.js, Remix, Svelte, Vue
